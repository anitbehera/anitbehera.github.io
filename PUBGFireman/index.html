Hello, squad!

I've reached out to PUBG developers about the loading smoke screen issue through both Reddit posts and official PUBG support forums(Ticket #19499453). However, since there's no communication for an official fix, I took matters into my own hands.

I've developed an application that resolves the loading smoke screen problem. You can see how it works in the demo video provided.

How it works ?

The app automatically detects and closes inactive connections to PUBG lobby servers, which eliminates the smoke screen loading issue.

Download & Requirements

The application is available for free download and use.

System Requirements: Windows App Runtime must be installed on your system.

Download PUBG Fireman

How to use?

  • Installation: Download the PUBGFireman.zip file from above link and extract it. Run 'PUBGFireman'.
  • Keep Running: Leave the application active throughout your PUBG gaming session (you can minimize it to run in the background).
  • Fix Smoke Screen: Before clicking Exit to Lobby in PUBG, press the Extinguisher button located in the bottom-left corner of PUBG.
  • Timing Note: While you can use the Extinguisher button at any time during the loading smoke screen, it's best to use it just before Exiting to Lobby since the effect takes 3-4 seconds to process.
